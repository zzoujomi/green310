flex_레이아웃속성

flex레이아웃은 부모자의 flex를 선언하고 자식요소를 유연하게 배치하는 속성

ul>li의 경우 ul에게 flex를 선언하면 li가 가로정렬됨
이때 li는 item의 속성을 사용한다.

flex _ container속성
속성명 속성값과 의미
display flex - 자손자를 flex로 정렬하겠다라는 선언문
justify-content 수평정렬에 관한 설정
  • felx-start : 시작점에서부터 정렬(기본값)
  • flex-end : 끝점에서 정렬
  • center : 아이템을 부모자의 가운데정렬
  • space-between : 처음아이템은 시작점에붙고 마지막아이템은 끝점에 붙인 상태에서 나머지 아이템을 사이정렬
  • space-around : 모든 아이템들의 좌우여백값 통일
aline-items 아이템들의 상하여백정렬(부모자의 높이가 지정되어있고 아이템들은 부모자 높이값보다 양이 적을때 사용)
  • strech : 아이템들의 위아래를 맞춤(기본값)
  • felx-start : 아이템들을 위로 붙임
  • flex-end : 아이템들을 아래로 붙임
  • center : 아이템을 부모기준으로 가운데 정렬
  • space-between : 처음아이템은 시작점에붙고 마지막아이템은 끝점에 붙인 상태에서 나머지 아이템을 사이정렬(상하기준)
  • space-around : 모든 아이템들의 상하여백값 통일
flex-direction column(위에서 아래로 수직정렬)
row(왼쪽에서 오른쪽으로 수평정렬)
reverse를 활용하여 역방향 가능
flex-wrap 기본적으로 flex는 자손자를 한줄에만 표시하려하므로 자손자의 크기는 무시하고 공간에 가둔다. 그래서 줄바꿈을 이루어 표기하려면 wrap으로 묶어준다. flex-wrap:wrap
wrap / no-wrap(기본값) / wrap-reverse
flex-item속성(자손자에 표기)
order item의 순서를 설정한다.
음수까지 허용하며 숫자가 높을수록 뒤로 밀림 / 기본값이 0
flex-grow 증가너비 비율
flex-shink 감소너비 비율 숫자가 크면 더 많은 너비가 감소한다.
flex-basis item 배분너비 기본값(auto),width로 잘 설정하지 않는다.
flex grow shink basis 순서로 기본값 0 0 auto
aline-self 아이템에게 각자지정이 가능하며 aline-item의 세분화작업 aline-item의 값을 재세팅가능하다.
