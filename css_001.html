css란?

cascading style sheet 약자로 문서를 꾸미는 구조로 사용된다.

css종류

  1. 외부스타일시트 - html문서 내부에 쓰지않고 별도의 css라는 확장자 이름을 가진 파일을 작성하고 link태그로 연결한다.
    작성법 link href="스타일시트.css"
  2. 내부스타일시트 - html문서 내부에 style태그를 만들어 작성한다. (에전엔 head태그 내부에 작성한다. 였지만 지금은 어디에 넣어도 상관없음)
  3. inline방식 - 태그에 속성값으로 직접 작성하는 방식(우선적용)
    작성법 : 태그명 style = "color(속성명) : red(속성값)"
