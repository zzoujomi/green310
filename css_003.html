CSS레이아웃

css를 활용하여 공간을 나누듯 배치를 하기위해 여러속성을 사용한다.

가시속성

display : 태그가 화면에 보이는 방식을 지정한다.

  • none - 화면에서 안보이며 공간도 함께 사라진다. 또한 웹접근성으로도 확인할 수 없다.
  • ★inline - 특별한 공간의 값을 가지고 있지않다. width,height등의 값을 지정할 수 없다. marjin, padding의 경우 요소를 기준으로 하기 때문에 좌우는 적용이 되나, 상하는 적용되지 않는다.
  • inline-block - inline과 블럭의 중간역할 두속성의 모든 성질을 가지고있음
  • block - 가로값 화면기준 100%적용 / 자동 줄바꿈 기능이있다.

float속성

자동줄바꿈의 성질이 있는 블럭을 한줄에 두개이상의 요소를 배치하고자 할 때 사용/ 사전적의미로 물이나 공중에 뜨다 라는 의미로 블럭요소를 ★강제로 띄워서 화면에 좌우로 배치

  • left : 왼쪽 요소를 띄워서 나열한다
  • right : 오른쪽 요소를 띄워서 나열한다.

flex속성

태그를 수평,수직으로 자유롭고 유연하게 변동할 수 있는 기능

  • container - 부모요소를 부르는 명칭으로 사용
  • items - 자식요소를 부르는 명칭으로 사용

container 속성에는 display, flex-flow(묶음기능), flex-derection(수평,수직), justify-content(아이템수평정렬), aline-items(아이템수직정렬)

items속성에는 flex, grow(증가너비),shrink(감소너비),basis(크기배분), align-self(각자조정)

자손자에만 오롯이 적용됨. 후손자 적용 안됨
