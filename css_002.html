박스속성

웹페이지 레이아웃시 가장 중요한 속성

  • 요소크기설정 - width, height
  • 영역설정 - padding
  • 외곽선설정 - border
  • 여백설정 - margin

작성법

  • 전체적용 - 숫자와 단위를 함께 하나만 작성
  • 각자적용 - 숫자와 단위를 4번 써서 작성한다. 순서는 위, 오른쪽, 아래, 왼쪽(시계방향)
    작성시 쉼표를 쓰지 않고 띄어쓰기를 통해 작성한다.
  • 세트적용 - 위아래/왼쪽 오른쪽 숫자와 단위를 2번써서 작성한다.
    앞에가 위아래, 뒤에가 왼쪽 오른쪽 각자 적용과 마찬가지로 띄어쓰기를 통해 작성한다.
  • 부분적용
    • margin-top:위쪽여백
    • margin-bottom : 아래쪽여백
    • margin-left : 왼쪽여백
    • margin-right : 오른쪽여백
    • padding-top:위쪽영역
    • padding-bottom:아래쪽영역
    • padding-left:왼쪽영역
    • padding-right:오른쪽영역

border설정방법

박스의 테두리를 말하며 패딩(padding)바깥쪽에 위치한다.
색상,굵기,모양으로 설정한다.

  • border-color : 색상기재
  • border-width : 숫자와 단위
  • border-style
    • none - 선을 없앰
    • solid - 실선
    • dotted - 동그라미점으로 점선만들기
    • dashed - 긴사각형 점선
    • double - 실선두줄
    • groove - (액자형) - 테두리가 파인것처럼 그려짐
    • ridge - (액자형) - 테두리가 튀어나온것처럼 그려짐(groove 반대스타일)
    • inset - (액자형) - 내부요소가 파인것처럼 그려짐
    • outset -(액자형) - 내부요소가 튀어나온것처럼 그려짐(inset반대스타일)

작성법 - color, width, style을 한번에 쓸수도 있고 각자 쓸수도 있다 순번에 관계없음
border : ridge 2px solod(2px 너비의 빨간색실선)

백그라운드(배경설정)

배경색이나 배경 이미지를 삽입할때 사용한다. 많은 속성들을 한 번에 기재할 수도 있고, 별도로 기재할 수 있다

  • background-color
    • 영문이름으로 작성 : red,black..등
    • 텍스트로 쓰는 방법 : #ffffff #000000 #fff 반복으로 들어갈 경우 레드에서 하나 그린에서하나 블루에서 하나씩 작성가능
    • rgb 수치값작성 : rgb(00,00,00)
    • rgb와 투명도를 같이 작성 : rgba(0,200,50,0.5)
  • background-image : url(이미지 경로작성)
  • background-repeat : 배경이미지 또는 컬러를 반복한다.
    • repeat : 가로, 세로형으로 모두 패턴처럼 깔림
    • repeat-x : 가로방향으로만 깔림
    • repeat-y : 세로방향으로만 깔림
    • no-repeat : 반복없음
  • background-size : 배경 이미지의 크기를 태그 사이즈대비 변경하고자 할 때 사용
    • contain : 이미지의 비율을 유지하면서 영역을 꽉 채움
    • cover : 배경이미지가 짤리더라도 화면의 영역을 꽉채움
    • background-position : top, center, legt 지칭하는 단어로도 사용하고 숫자와 단위로도 이동가능하다.
      왼쪽 상단에서부터 거리가 멀어지고 앞수가 오른쪽으로 이동. 다음수가 아래쪽으로 이동(20px 60px)
    • background-attachment : fixd로 바닥에 붙이는 효과

그라디언트를 활용하며 배경색 또는 패턴을 제작할 수 있다.

  • linear-gradient(각도,시작색깔,마지막색깔)
  • radial-gradient(시작색깔,마지막색깔)
  • patten - 직선그라데이션의 수직 시작점을 %값으로 잡아 옆에 여백을 만들어 반복시켜 적용
  • 그라디언트헌트바로가기
