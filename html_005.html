입력양식태그

입력양식은 사용자에 정보를 입력받거나 보내는 요소. 예를들어 로그인/회원가입/검색등이 있다.

form태그내부에 작성하며 다양한 방식의 요소를 활용한다.

form태그 내부에 전달방식을 기재하는 method요소를 활용한다.

  • get방식 - 직접전달(오픈형 )/당근마켓
  • post방식 - 비밀전달 (폐쇄형)/본인외개봉금지

입력양식태그

태그명 속성명 (type="속성값") 설명
form 특성 : 보이지않음 / block태그 입력양식의 처음과 끝
input - inline양식 text 글자입력양식
password 비밀번호양식 / *로 표기됨
submit 제출버튼(확인)
reset 초기화버튼
file 내 컴퓨터에 있는 폴더열기
search 검색박스로 외관으로는 text 박스와 같으나 끝에X표시 나타남
button 버튼생성/
checkbox 다중선택박스- 사각형모양

email @ 를 넣지않았을경우 입력양식오류 나타남
number 숫자입력방식 - 다양한속성을 함께작성
color 컬러를 선택할수 있는 기능

min="최소값" max='최대값' step='건너띄기' 수가 들어가는 경우 모두 적용

select multipul='multipul' 선택양식생성
option 셀렉트태그 내부에 선택하는요소
optgroup option태그를 묶어주는 역할 / 클릭은안됨
fieldset 그룹된 부분을 선으로 그래서 박스를 만들어준다.
legend fieldset 내부에 선을 뚫고 써지는 제목
