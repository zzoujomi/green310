제이쿼리란?

제이쿼리는 2006년 발표된 자바스크립트 라이브러리로 무료로 사용 가능한 오픈소스라이브러리이다. 특징은 다음과 같다.

  • 쉬운 문서 객체 모델과 관련한 처리 구현
  • 쉽고 일관된 이벤트 연결 구현
  • 쉬운 시각적 효과 구현

작성법 : CDN을 연결하고 CDN연결 태그 하단에 스크립트를 작성한다.

자바스크립트 : window.onload와 같은 의미로 $(document).ready(function(){실행문})의 구조로 작성되며
$(function){ 실행문
}) 으로 축약해서도 사용한다

제이쿼리탐색구문

선택자의 디테일한 부분으로 자손,부모,형제,순번 등에 활용된 형태

  • $('선택자').eq() : 선택자의 순번의 의미로 0부터 차례대로 번호를 부른다.
  • $('선택자').children() : 선택자의 자손의 의미로 해당태그 또는 클레스명 등 부른다.
  • $('선택자').parent() : 선택자의 부모의 의미로 태그명을 붙이거나 생략 가능하다.
  • $('선택자').siblings() :선택자의 형제의 의미로 본인은 제외된다.
  • $('선택자').find() : 선택자의 후손의 의미로 자손 내부에 있는 요소 모두를 부를 수 있다.
  • $('선택자').next() : 선택자의 다음 요소
  • $('선택자').prev() : 선택자의 이전 요소
  • $('선택자').not() : 선택자의 제외 요소 not안에 요소이름을 작성
