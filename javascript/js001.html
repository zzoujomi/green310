자바스크립트_연산자

설명문은 document에서 확인 할 수 있고 자바스크립트 예제는 console에서 확인할 수 있다.

산술연산자

  • 더하기 : +
  • 빼기 : -
  • 곱하기 : *
  • 나누기 : /
  • 나머지 : %

작성법
변수로 숫자를 지정한다.
let 또는 var 또는 const로 변수를 말한다.
let - 가장 유연하게 사용할 수 있고 모든 변수 이름 앞에 사용할 수 있다.
에러가 가장 나지 않음.
var - 처음으로 생긴 변수 지정법으로 가장 오래된 역사를 가지고 있으며 많은 오픈 파일에서 볼 수 있다.
단점은 중복되더라도 발견하지 못한다.
const - 고유 명사같은 개념으로 변수를 바꿀 수 없다.
대부분 선택자의 이름으로 한번 할당되면 바뀌지 않는 값에 쓰인다.
let a = 10
풀이 -> a는 10이다.
let b = 5 풀이 -> b는 5이다.
let c;
풀이 -> c를 변수 이름으로 사용하겠다.
c=a+b
풀이 -> c는 10+5이다.

비교연산자

변수가 두가지가 필요하며 두개의 데이터를 부등호를 통해 비교하며 바르면 true 틀리면 false가 출력된다.

  • a가 b보다 크다 - a>b
  • a가 b보다 크거나 같다 - a>=b
  • a와 b와 같다 - a==b
  • a가 b와 다르다 - a!=b
  • a가 50보다 크거나 20보다 작다
    -a>50&&a<20< /li>

논리연산자

  • 그리고(and) - &&
  • 또한(or) - \\
  • 아님(not) - !

예제풀이

나이를 물은 후 20세 이상이면 true 이하이면 false가 나타나게해라. 만약에 age가 20보다 크면 '당신은 성인입니다.'
라는 말을 document에 출력하고
그렇지 않으면 '당신은 성인이 아닙니다.'

증감연산자

변수에 저장된 값에 1씩 증가하거나 감소한다.

  • 증가 : ++
  • 감소 : --

대입연산자

연산된 데이터를 지정된 변수에 대입한다.

  • = 은 뒤에 있는 숫자나 내용을 앞의 변수에 대입한다.
  • 변수 =5
    풀이 5는 변수다.
    변수 +=5
    풀이 5를 더해서 변수에 대입해라.
